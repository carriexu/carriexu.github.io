Web Design for the Completely Clueless

Nov 22nd, 2014 11:51 pm

Larry Buchanan volunteered 3 hours of his time today and talked to us about design.

The design process is a complete mystery to someone who hasn’t studied design before, namely me. Larry highlighted some key design principles and walked us through his design process using an example. Thank you Larry.

A few of my takeaways:

  • Sketch first, code later. Thinking about design while coding is actually slower than if we sketched first and turn it into code later

  • Color

    • RGB is for web and CMYK is for print, when using design software such as Adobe Illustrator and Indesign, always remember to switch to RGB when we want to deploy to the web
    • To start: use black, white, and one other color, dark gray works well to bring up the sophistication feel of the site as pure black tend to overwhelm

  • Typography

    • Serif and Sans Serif, try one of each to bring out contrast
    • Consistency builds street cred

  • Layout

    • Make something dominant
    • Use the Grid system like Skeleton to help make sites responsive
